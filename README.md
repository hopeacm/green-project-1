环境配置：
    1、安装yarn
        sudo npm install -g yarn 或 yarn install
    2、查看是否安装成功： yarn -v
    3、在项目目录下终端输入：
        yarn init
        yarn add jquery
        yarn start 执行
    4、本地服务器配置完成
